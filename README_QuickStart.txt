============================================
Ultimate PDF Editor - Quick Start
============================================

1. Launch the application:
   - Double-click PDFEditor.exe (or pdf_editor_exe_ready.exe).

2. Open a PDF:
   - Click "Open PDF" on the toolbar and select a file.

3. Editing Tools:
   - Pen: Draw freehand on pages.
   - Eraser: Remove drawings.
   - Add Text: Insert text anywhere.
   - Clear All: Remove all edits.

4. Zoom:
   - Use "Zoom In" / "Zoom Out" for detailed editing.

5. Save:
   - Click "Save PDF" to save edits as a new file.

6. Exit:
   - Click "Exit" to close the application.

Tips:
- Page numbers are displayed on each page.
- Scroll to view all pages.
