--------------------------------------------
Description:
--------------------------------------------
Ultimate PDF Editor is a lightweight, standalone PDF editor built with Python and PyQt5. 
It allows you to open, edit, and save PDFs efficiently, with a modern and easy-to-use interface. 
This editor supports lazy-loading of pages for large PDFs, ensuring smooth performance.

--------------------------------------------
Features:
--------------------------------------------
1. Open PDF files of any size.
2. Scrollable multi-page view with lazy-loading.
3. Pen/Freehand drawing on pages.
4. Eraser tool to remove drawings.
5. Add text anywhere on the PDF.
6. Clear all edits on all pages.
7. Zoom in and zoom out for detailed editing.
8. Save edited PDF as a new file.
9. Exit the application easily.
10. Page numbers displayed on each page.
11. Modern GUI with shadows and responsive layout.
12. Fully standalone EXE version (no Python installation required).

--------------------------------------------
Usage Instructions:
--------------------------------------------
1. Launch the application (PDFEditor.exe or pdf_editor_exe_ready.exe).
2. Use the toolbar at the top to select the tool:
   - Open PDF: Load a PDF file to edit.
   - Save PDF: Save your edited PDF to a new file.
   - Exit: Close the application.
   - Pen: Draw freehand on the PDF.
   - Eraser: Erase drawings.
   - Add Text: Add text to a page.
   - Clear All: Remove all drawings and text.
   - Zoom In/Out: Adjust page zoom for better visibility.
3. Scroll through pages using the scrollbar.
4. Your edits will appear live on the pages.
5. After editing, click Save PDF to create a new edited PDF.

--------------------------------------------
System Requirements:
--------------------------------------------
- Windows 7 / 8 / 10 / 11
- 64-bit recommended
- No Python installation required for EXE version
- For Python version: Requires Python 3.9+, PyQt5, PyMuPDF (fitz), Pillow

--------------------------------------------
Distribution:
--------------------------------------------
- You can distribute the EXE standalone version located in the `dist/` folder.
- Optional: Use an installer (e.g., Inno Setup) for professional packaging.
- Include this README with your distribution to guide users.

--------------------------------------------
Notes:
--------------------------------------------
- The editor uses a text-only toolbar to avoid dependency on external icons.
- Lazy-loading ensures fast performance for PDFs with many pages.
- Saved PDFs preserve your edits as new files, original PDFs remain unchanged.
